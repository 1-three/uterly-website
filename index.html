Methodology
How Uterly works

A four-stage pipeline from raw ultrasound image to explainable clinical prediction.

01
🖼️
4-Channel Input
Each ultrasound is loaded as 224×224 RGB, then a Sobel edge-detection filter is applied to produce a 4th channel capturing follicle boundaries and structural detail invisible to colour-only models.
02
🧠
Dual-Path Feature Extraction
The RGB channels pass through EfficientNetB3 (pretrained on ImageNet, last 20 layers fine-tuned). The edge channel takes a separate Conv2D branch. Both extract complementary features.
03
🔗
Feature Fusion
A 512-dim RGB feature vector and a 64-dim edge feature vector are concatenated into a 576-dim joint representation, then passed through Dense(256) → Dense(2, softmax) classification layers.
04
🗺️
Grad-CAM Explainability
Gradients of the predicted class are backpropagated to the last convolutional layer. Channel-weighted feature maps produce a spatial heatmap showing exactly where the model "looked" to make its decision.
